## Installation

```shell
composer require snicco/pimple-bridge
```

## Usage

```php
use Pimple\Container;
use Snicco\Bridge\Pimple\PimpleContainerAdapter;

$pimple_container_adapter = new PimpleContainerAdapter(
    new Container(); // Your pimple container
)
```
